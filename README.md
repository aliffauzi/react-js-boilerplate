# About
React Js Boilerplate is a React js template to create web and make the development process even faster.

## Installation

Clone this repo

```bash
https://github.com/kubil-ismail/react-js-boilerplate.git
```
Intall dependency.

```bash
cd react-js-boilerplate
npm install
npm start
```
